# HuffmanCode
A simple implementation of HuffmanCode

## How to build
```
git clone github.com/bogdasar1985/HuffmanCode.git

cd HuffmanCode

mkdir build

cd build

cmake ..

make -j4
```

## How to use
```bash
./coder <decompressed-file> <compressed-file>

./decoder <compressed-file> <decompressed-file>
```
